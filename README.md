## 📱 About Hirely

Hirely is a user-friendly job portal application designed to bridge the gap between job seekers and employers. The application allows users to create accounts, post job opportunities, browse available jobs, view detailed job information, and connect with recruiters seamlessly.

The project was developed as an academic Android application focusing on modern UI design, cloud-based data management, and real-time job listing functionality.
